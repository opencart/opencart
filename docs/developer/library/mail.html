Mail

\Opencart\System\Library\Mail sends email through a pluggable adaptor: PHP's mail() or SMTP.

Adaptors

EngineFileNotes
mailsystem/library/mail/mail.phpDefault. Uses PHP's built-in mail() -- configured by the host's MTA.
smtpsystem/library/mail/smtp.phpDirect SMTP. Reads host, port, username, password and encryption mode from $option.

The active engine is selected by System → Settings → Mail (config_mail_engine).

Constructor

```php $mail = new \Opencart\System\Library\Mail($this->config->get('config_mail_engine'), [ 'smtp_hostname' => $this->config->get('config_mail_smtp_hostname'), 'smtp_username' => $this->config->get('config_mail_smtp_username'), 'smtp_password' => $this->config->get('config_mail_smtp_password'), 'smtp_port' => $this->config->get('config_mail_smtp_port'), 'smtp_timeout' => $this->config->get('config_mail_smtp_timeout'), ]); ```

API

MethodDescription
setTo(string|array $to)Single address or list.
setFrom(string $from)Envelope sender.
setSender(string $sender)Display name.
setReplyTo(string $reply_to)Reply-To header.
setSubject(string $subject)Subject line.
setText(string $text)Plain-text body.
setHtml(string $html)HTML body. Pair with setText() for multipart.
addAttachment(string $filename)Absolute path to a file to attach.
send(): boolSends. Returns true on success.

Usage

```php $mail->setTo($customer_email); $mail->setFrom($this->config->get('config_email')); $mail->setSender(html_entity_decode($store_name, ENT_QUOTES, 'UTF-8')); $mail->setSubject(html_entity_decode($subject, ENT_QUOTES, 'UTF-8')); $mail->setHtml($html_body); $mail->setText(strip_tags($html_body)); $mail->send(); ```

HTML entities

Strings that came through the database often arrive HTML-encoded. Decode with html_entity_decode(..., ENT_QUOTES, 'UTF-8') before passing to setSubject / setSender -- otherwise recipients see raw &amp;.
