Document

\Opencart\System\Library\Document collects everything the layout needs to print into <head> and the script slots: title, description, keywords, link/canonical tags, stylesheets, scripts and meta tags.

API

MethodDescription
setTitle(string $title) / getTitle()The <title> tag.
setDescription(string $description) / getDescription()The meta description.
setKeywords(string $keywords) / getKeywords()The meta keywords.
addLink(string $href, string $rel) / getLinks()<link rel="canonical">, alternate, etc.
addStyle(string $href, string $rel = 'stylesheet', string $media = 'screen') / getStyles()External stylesheets.
addScript(string $href, string $position = 'header') / getScripts(string $position = 'header')External scripts. $position is either header or footer.
addMeta(string $meta) / getMeta()Raw meta tag entries (full HTML).

Usage

From a controller:

```php $this->document->setTitle($this->language->get('heading_title')); $this->document->setDescription($this->language->get('meta_description')); $this->document->addLink('https://example.com/canonical', 'canonical'); $this->document->addStyle('view/stylesheet/extension/hello.css'); $this->document->addScript('view/javascript/extension/hello.js', 'footer'); ```

The base layout (common/header / common/footer) iterates over these and prints them. Adding the same href twice produces duplicate tags; deduplicate at your callsite if needed.

SEO link helpers

For canonical URLs that point at a route inside the same store, build the href via $this->url->link(...):

```php $this->document->addLink($this->url->link('product/product', 'product_id=' . $product_id), 'canonical'); ```
