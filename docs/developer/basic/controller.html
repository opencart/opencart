Controllers

A controller is the entry point for a route. The router resolves a request like ?route=common/dashboard to a controller class, instantiates it, and calls the requested method.

Where controllers live

PathUsed by

catalog/controller/<group>/<name>.phpCore storefront controllers.
extension/<code>/admin/controller/<group>/<name>.phpAdmin controllers shipped by an extension.
extension/<code>/catalog/controller/<group>/<name>.phpCatalog controllers shipped by an extension.

Namespace and class name

The autoloader maps Opencart\Admin to admin/, Opencart\Catalog to catalog/ and Opencart\Extension to the extension storage. The route, namespace and file name must agree:

RouteFileClass
common/dashboardadmin/controller/common/dashboard.phpOpencart\Admin\Controller\Common\Dashboard
extension/hello/module/helloextension/hello/admin/controller/module/hello.phpOpencart\Admin\Controller\Extension\Hello\Module\Hello

Snake-case route segments become StudlyCase namespace segments (my_module → MyModule).

Methods and the default action

If the route has a trailing .method, that method is called; otherwise index() is called. So:

  • ?route=catalog/product → Product::index()
  • ?route=catalog/product.save → Product::save()

Methods starting with __ are blocked by the loader to keep magic methods unreachable from the URL.

What is available on $this

Controllers extend \Opencart\System\Engine\Controller, which forwards property access to the registry. The most common services:

PropertyClassNotes
$this->loadLoaderLoads other controllers, models, views, languages, libraries.
$this->configConfigRead store settings and runtime config.
$this->dbDBDatabase query interface.
$this->sessionSessionServer-side session storage.
$this->requestRequestget, post, cookie, files, server.
$this->responseResponseHeaders, body, redirect.
$this->languageLanguageTranslation strings loaded by $this->load->language(...).
$this->urlUrlBuild links with link($route, $args, $secure).
$this->documentDocumentPage title, meta, links, scripts.
$this->cacheCacheApplication cache.
$this->userUser (admin only)Currently authenticated admin user.
$this->customerCustomer (catalog only)Currently authenticated storefront customer.

Loading another controller

To compose a page from sub-controllers, use the loader:

```php $data['header'] = $this->load->controller('common/header'); $data['column_left'] = $this->load->controller('common/column_left'); $data['footer'] = $this->load->controller('common/footer'); ```

The return value is whatever the called method returned (typically the rendered HTML of a partial view).

Returning output

A controller does not return its HTML directly. It writes to the response object:

```php $this->response->setOutput($this->load->view('common/dashboard', $data)); ```

For JSON endpoints (e.g. .save handlers):

```php $this->response->addHeader('Content-Type: application/json'); $this->response->setOutput(json_encode($json)); ```

For redirects:

```php $this->response->redirect($this->url->link('common/dashboard', 'user_token=' . $this->session->data['user_token'])); ```
