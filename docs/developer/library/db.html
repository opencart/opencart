DB

\Opencart\System\Library\DB is the database facade. It picks an adaptor at construction time and forwards every call. Models talk to it through $this->db.

Adaptors

EngineFileNotes
mysqlisystem/library/db/mysqli.phpDefault. Procedural mysqli with tracking of affected rows / last id.
pdosystem/library/db/pdo.phpPDO MySQL driver.
pgsqlsystem/library/db/pgsql.phpPostgreSQL via the pgsql extension.

The active adaptor is configured by the db_engine key in config.php.

API

MethodDescription
query(string $sql): objectRuns a query. SELECT returns an object with num_rows, row (first row) and rows (all rows). Other statements return a generic object (used for chained metadata).
escape(string $value): stringQuotes a string for inclusion in a query. Does not add the surrounding quotes.
countAffected(): intAffected rows from the last write.
getLastId(): intThe auto-increment value of the last insert.
isConnected(): boolWhether the underlying connection is alive.

Reading rows

```php $query = $this->db->query("SELECT * FROM `" . DB_PREFIX . "product` WHERE `status` = '1'"); foreach ($query->rows as $row) { // ... } $first = $query->row; $count = $query->num_rows; ```

Writing safely

OpenCart does not expose prepared statements at this layer. Build queries by hand and protect every value:

  • Cast scalars: (int)$id, (float)$price.
  • Wrap strings: '" . $this->db->escape($value) . "'.
  • Always prefix table names with DB_PREFIX.
  • Backtick identifiers (column / table names) so reserved words stay parseable.
```php $this->db->query("INSERT INTO `" . DB_PREFIX . "log` SET `message` = '" . $this->db->escape($message) . "', `created_at` = NOW()"); $id = $this->db->getLastId(); ```

Time zones

cron.php and system/framework.php both run a SET time_zone = '...' immediately after connecting so PHP and the database agree on the offset. If you switch the connection at runtime, repeat that step.
